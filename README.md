# Appointment Booking System

An online appointment scheduling platform designed to automate bookings, manage staff calendars, accept payments, and improve customer engagement.

It enables businesses to offer 24/7 online booking while reducing manual scheduling tasks.

## Included UI
- `index.html`: Common responsive page for web and app users to self-book appointments.
- `styles.css`: Shared styles for consistent cross-device experience.

## Core Capabilities
- Online self-service appointment booking
- Staff calendar and availability management
- Integrated payment acceptance
- Customer engagement workflows and reminders
- Reduced administrative overhead through scheduling automation
